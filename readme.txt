Description
--------------------------------------------

Rex is a multipurpose landing page template which designed for any kind of businesses and startups. Rex is fully responsive which means it will automatically optimize in any kind of devices like mobile, tablet, phablet, laptop, desktop etc.

Key Features
=============

* HTML5 and CSS3 Validated
* Responsive Template
* Content Slider
* Slick Slider
* Single Page Template
* Built on Bootstrap 3.3.4
* 479 icons ( based on Font Awesome )
* 10 Themes color variations
* 11 Section Feature in Home page
* Slider Section
* About Section
* Service Section
* Portfolio Section
* Price Section
* Testimonial Section with Slider
* Blog Section
* Client Feature Section
* Blog Archive Page
* Blog Single Page
* 404 Error Page
* Contact
* Google Maps integration
* Google Web Fonts

& lots more
